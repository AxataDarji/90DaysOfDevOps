## GitHub-Hosted vs Self-Hosted Runners

| Feature | GitHub-Hosted | Self-Hosted |
|-------- |---------------|-------------|
| **Who manages it?** | GitHub | You / your team |
| **Cost** | Free with usage limits (included in GitHub plans) | You pay for your own machine / VM |
| **Pre-installed tools** | Many tools pre-installed (Docker, Node, Python, Git, etc.) | You must install and manage tools yourself |
| **Good for** | Quick CI jobs, testing across OS (Linux, Windows, macOS), temporary workloads | Long-running jobs, custom environments, heavy builds |
| **Security concern** | Runs on shared infrastructure managed by GitHub | You are responsible for securing the machine and updates |
